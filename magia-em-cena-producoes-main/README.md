# Magia em Cena Produções

Uma landing page moderna, encantadora e totalmente responsiva desenvolvida para uma empresa de animação de festas infantis. O objetivo do projeto é apresentar os serviços, portfólio de personagens e depoimentos da empresa com uma interface fluida e interativa.

## Funcionalidades e Destaques

* **Design Responsivo:** Layout cuidadosamente adaptado para oferecer a melhor experiência em celulares, tablets e computadores.
* **Carrossel 3D (Coverflow):** Galeria interativa de personagens com efeito de profundidade, utilizando a biblioteca Swiper.js.
* **Animações de Scroll:** Elementos e textos que surgem suavemente na tela conforme a rolagem da página, utilizando ScrollReveal.
* **Captação de Orçamentos:** Modal de formulário dinâmico que coleta os dados do cliente e envia uma mensagem formatada diretamente para o WhatsApp da empresa.
* **Acessibilidade:** Integração com a ferramenta VLibras para acessibilidade de tradução.

## Tecnologias Utilizadas

* **HTML5 & CSS3** (Flexbox, CSS Variables, Efeitos visuais e Gradientes)
* **JavaScript (Vanilla)**
* [Swiper.js](https://swiperjs.com/) para os carrosséis
* [ScrollReveal](https://scrollrevealjs.org/) para as animações de entrada
* [FontAwesome](https://fontawesome.com/) para a iconografia
